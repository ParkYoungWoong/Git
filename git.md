# git

## How to use

프로젝트 생성

```
git init
git add 파일명
git commit
```

```
git add 파일명
git commit
ESC >> i(INSERT) >> 커밋 메세지 입력
ESC >> : >> wq(저장 후 종료) >> 엔터
git log
```

```
git config --global user.name 이름
git config --global user.email 이메일
```
